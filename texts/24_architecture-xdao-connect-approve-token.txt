Approve Token
To make operations with tokens, you need to approve to interact with them. You can do this either on the site itself by connecting with the DAO wallet via WalletConnect, or by creating a voting.
On the DAO home page select the "Create Voting" button.
Select "Approve Token" from the drop-down list.
You can get the address of the required token for the approve on coinmarketcap or coingecko.
The Spender address for the well-known sites is listed below in the table.
We recommend setting a larger Token Amount, for example, 1000000000000000.
